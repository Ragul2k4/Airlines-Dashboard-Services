# Airline Dashboard Services

## Overview
The Airline Dashboard Services application is designed to provide real-time management and display of flight schedules. Developed using HTML, CSS, JavaScript, and MongoDB, the application allows users to view flight details, including flight time, name, and gate information, as well as manage flight records by adding, updating, and deleting entries.

## Features
- **Real-Time Flight Information:** View up-to-date flight schedules with essential details.
- **Flight Data Management:** Insert, update, and delete flight records.
- **Responsive Design:** Accessible across various devices, including desktops, tablets, and mobile phones.

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MongoDB
